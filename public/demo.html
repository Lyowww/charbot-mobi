How to Embed the Chatbot Widget

Simply add this script tag to any website:

<script src="https://your-domain.com/chatbot-widget.js"></script>

Or with custom configuration:

<script src="https://your-domain.com/chatbot-widget.js" data-position="bottom-left" data-primary-color="#28a745" data-width="350px" data-height="450px"></script>

Advanced Configuration

For more control, initialize the widget manually:

<script src="https://your-domain.com/chatbot-widget.js" data-auto-init="false"></script> <script> ChatbotWidget.init({ position: 'bottom-right', primaryColor: '#007bff', backgroundColor: '#ffffff', textColor: '#333333', width: '300px', height: '400px', borderRadius: '8px', zIndex: 9999 }); </script>

Widget API

Control the widget programmatically:

// Open the chat ChatbotWidget.open(); // Close the chat ChatbotWidget.close(); // Send a message programmatically ChatbotWidget.sendMessage('Hello from JavaScript!'); // Update configuration ChatbotWidget.config({ primaryColor: '#ff6b6b' });

Deployment Instructions

  1. Upload the chatbot-widget.js file to your web server or CDN
  2. Update the script src URL to point to your hosted file
  3. Add the script tag to any website where you want the chatbot
  4. Customize the appearance using data attributes or the API
